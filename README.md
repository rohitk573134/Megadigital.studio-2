# 📸 Mega Digital Studio - Premium Photography Website

A luxury, cinematic 3D photography studio website built with modern web technologies.

![Mega Digital Studio](https://images.unsplash.com/photo-1519741497674-611481863552?w=1200&q=80)

## ✨ Features

- 🎬 **Cinematic Preloader** - Animated camera lens with progress bar
- 🔮 **3D Camera Lens** - Interactive Three.js lens with mouse tracking
- ✨ **Floating Particles** - Gold animated particles in hero section
- 📜 **Scroll Story Experience** - 4 immersive story sections
- 🃏 **3D Service Cards** - 6 premium cards with tilt effect
- 🖼️ **Portfolio Gallery** - Masonry grid with category filtering
- 💬 **Testimonials Carousel** - 3D rotating carousel with touch support
- 📱 **Fully Responsive** - Optimized for all devices
- ⚡ **Performance Optimized** - Lazy loading, throttling, 60fps animations

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Grid, Flexbox, Animations
- **JavaScript** - ES6+ Vanilla JS
- **Three.js** - 3D graphics
- **GSAP** - Animations & ScrollTrigger
- **Google Fonts** - Cormorant Garamond & Inter
